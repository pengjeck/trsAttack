# desktop app with electron\vue\vuex

> A desktop app 

## Build Setup

``` bash
### install dependencies
npm install

### serve with hot reload at localhost:8080
npm run dev

### run app(after server started)
> you should install electron before
> $npm install electron -g
cd project_path
electron .
```
